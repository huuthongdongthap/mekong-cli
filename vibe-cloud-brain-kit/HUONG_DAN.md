## 🚀 CÁCH CÀI ĐẶT (WINDOWS / MAC / LINUX)

### BƯỚC 1: Giải nén
Bạn đã tải file `vibe-cloud-brain-v1.zip` về. Hãy giải nén nó ra.

### BƯỚC 2: Chạy File Cài Đặt
- **Mac/Linux:** Mở Terminal, gõ lệnh:
  ```bash
  chmod +x cloud-brain.sh
  ./cloud-brain.sh
  ```
- **Windows:** (Sắp ra mắt, dùng WSL tạm nhé).

### BƯỚC 3: Làm Theo Hướng Dẫn Trên Màn Hình
1. Script sẽ tự động mở **Google Colab**.
2. Bạn chỉ cần **Ctrl+V** (Dán code) -> Bấm **Play (▶️)** trên Colab.
3. Chờ hiện Link thần thánh -> Copy dán vào Terminal -> **Enter**.
