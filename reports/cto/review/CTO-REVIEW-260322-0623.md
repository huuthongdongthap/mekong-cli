## CTO Daemon v7.5 Improvements

### New Features

1. **Stricter Error Detection**
   - Ignores shell noise (exit codes, permission denied, etc.)
   - Requires 2+ real error lines to trigger
   - No false positives from "0 errors" messages

2. **Override Attempt Limiting**
   - Max 2 attempts per category (errors/tests/conflicts)
   - Skips to productive work after exhaustion
   - Prevents infinite loops on stuck workers

3. **Heartbeat Monitoring**
   - Writes to `~/.openclaw/HEARTBEAT.md` every cycle
   - Chairman can monitor daemon health
   - Includes worker status and context %

4. **Log Rotation**
   - Auto-rotates when `~/.mekong/cto-daemon-stderr.log` > 5MB
   - Prevents disk space exhaustion

5. **Enhanced Question Detection**
   - Better pattern matching for confirmations
   - Smart answer logic based on question type
   - Auto-dismisses menu prompts

### Code Changes

- 171 insertions, 189 deletions
- Net: -18 lines (more efficient)
- Added: `shutil` import for log rotation
- Added: `HEARTBEAT_FILE`, `LOG_FILE` constants
- Added: `SHELL_ERROR_NOISE` pattern list
