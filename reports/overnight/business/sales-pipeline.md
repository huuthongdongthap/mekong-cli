# Sales Pipeline & Process — Mekong CLI

## Overview
Mekong CLI operates a hybrid sales motion: self-serve PLG for Starter/Pro tiers and a light-touch SDR/AE process for Enterprise. The full sales pipeline is managed and executed by Mekong CLI commands themselves — dogfooding the product.

---

| Tier | Price | Sales Motion | Human Touch |
|------|-------|-------------|-------------|
| Starter $49 | Self-serve | PLG → in-app upgrade | None |
| Pro $149 | Self-serve + email | PLG + activation email | Minimal |
| Enterprise $499 | Assisted | SDR qualify → AE close | High |

## Pipeline Stages

```
Stage 1: PROSPECT     → Identify ICP accounts (mekong sdr-prospect)
Stage 2: OUTREACH     → Send personalized email/DM (mekong sdr-outreach-blast)
Stage 3: QUALIFY      → Discovery call or async qualify (mekong sdr-lead-qualify)
Stage 4: DEMO/PREP    → Prepare custom demo (mekong ae-deal-prep)
Stage 5: PROPOSAL     → Send proposal (mekong proposal)
Stage 6: FOLLOW-UP    → Nurture and handle objections (mekong ae-follow-up)
Stage 7: CLOSE        → Contract + Polar.sh payment (mekong ae-close-report)
Stage 8: ONBOARD      → Customer success handoff (mekong business-client-onboard)
```

## Ideal Customer Profile (ICP)

### Primary ICP: Technical Founder / Solo CTO
- Company stage: Pre-seed to Series A
- Team size: 1-20 people
- Pain: Overwhelmed with ops, limited budget for tools/staff
- Tech stack: Python, CLI-comfortable
- LLM familiarity: Has OpenAI or OpenRouter key
- Budget: $49-149/mo (Starter/Pro)
- Where to find: Hacker News, GitHub, IndieHackers, Twitter

### Secondary ICP: Agency Owner
- Company: Digital/dev agency, 5-50 employees
- Pain: Client delivery bottlenecks, high labor cost
- Need: White-label AI ops layer
- Budget: $149-499/mo (Pro/Enterprise)
- Where to find: LinkedIn, agency communities, Slack groups

### Tertiary ICP: Enterprise Engineering Team
- Company: 50-500 employees
- Pain: Internal tooling costs, developer productivity
- Need: On-premise LLM deployment + unlimited commands
- Budget: $499+/mo (Enterprise, possibly custom)
- Where to find: Outbound, conference sponsorship

## AE Workflow (commands: `mekong ae-deal-prep`, `mekong ae-follow-up`, `mekong ae-close-report`)

### Deal Preparation (`mekong ae-deal-prep`)
- Research prospect's company, stack, pain points
- Prepare custom demo: run their specific use case with Mekong CLI
- Prepare ROI calculation: their current tool cost vs $149/mo Pro
- Identify stakeholders: who signs, who influences

### Demo Script
1. Open terminal: `mekong cook "Build me a sales pipeline for a SaaS company"`
2. Show PEV: plan → execute → verify in real-time
3. Run `mekong finance-monthly-close` — watch it generate P&L draft
4. Run `mekong sdr-prospect` — watch it source leads from criteria
5. Close: "Which of your weekly tasks would you run first?"

### Follow-Up Cadence (`mekong ae-follow-up`)
- Day 1 post-demo: Summary + ROI doc
- Day 3: Case study link relevant to their vertical
- Day 7: Check-in + specific command suggestion for their pain
- Day 14: Final follow-up + expiring offer (20% off annual)

### Close Process (`mekong ae-close-report`)
- Send Polar.sh payment link directly
- Enterprise: custom contract via `mekong contract`
- Handoff to customer success: `mekong business-client-onboard`

| Qualify | 6/mo | 50% → Demo |

### Pipeline Health Indicators
- Days in stage: >14 days in Qualify = stuck, needs re-engage
- Response rate: <10% on outreach = messaging needs refresh
- Demo-to-close: <50% = demo script needs revision

## Lead Qualification Framework (command: `mekong sdr-lead-qualify`)

**MEDDIC adapted for Mekong CLI:**
- **M**etrics: What's their current ops cost? How many tools? How many hours/wk on ops?
- **E**conomic buyer: Founder or CTO? Do they have budget authority?
- **D**ecision criteria: Own LLM key (free tier) or managed (RaaS)?
- **D**ecision process: Solo decision or team approval?
- **I**dentify pain: Code ops bottleneck, business ops overhead, both?
- **C**hampion: Someone who's already used/tried it?

**Disqualify if:**
- No LLM API key and not interested in RaaS ($49+)
- Company uses only GUI tools, no CLI comfort
- Enterprise security requirements incompatible with LLM routing
