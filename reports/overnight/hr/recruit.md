## Role 1: Community Manager (First Hire)

### Job Description

**Title:** Community Manager — Mekong CLI / OpenClaw
**Type:** Full-time, remote-first
**Location:** Anywhere (async-first, Vietnamese timezone preferred)
**Compensation:** $60,000–$80,000 USD/year + equity (0.5–1.0%)
**Start:** Target Q3 2026 (at $5K MRR)

**About the Role**

You will be the first non-founder hire at Mekong CLI, the AI-operated business platform
built on Vietnamese 孫子兵法 strategy. You will own the developer community from day zero:
GitHub presence, Discord server, content pipeline, and first-response support.

**Responsibilities**

- Manage GitHub: triage issues, label bugs, respond to PRs within 24 hours
- Own Discord: moderate, answer questions, host weekly office hours
- Content: Write 2 blog posts/week (tutorials, use cases, changelog)
- Social: Twitter/X build-in-public thread, LinkedIn developer content
- Events: Coordinate Product Hunt launches, HackerNews Show HN posts
- Support: First-line customer support for Starter and Pro tiers (48-hour SLA)
- Metrics: Track stars, forks, Discord members, issue resolution time
- Feedback: Synthesize community feedback into weekly product report

**Requirements**

- 2+ years developer relations, community management, or developer advocacy
- Comfortable with Python-based CLI tools (can install, run, debug basic issues)
- Strong written English AND Vietnamese (bilingual preferred, English minimum)
- GitHub fluency: PRs, issues, labels, milestones, GitHub Discussions
- Async-first mindset: Notion, Linear, Discord over Slack/meetings
- Self-directed: Works independently across timezones

**Nice to Have**

- Experience with open source developer tools community
- YouTube/streaming experience for live coding sessions
- Understanding of LLM/AI tooling landscape
- Vietnamese tech community connections (WeBuild, Viblo, TopDev)

**What We Offer**

- Full remote, async-first culture
- Equity in a fast-growing open source project
- Direct impact: You are hire #1, shape the culture
- Learning budget: $1,000/year for conferences, courses
- Equipment budget: $1,500 one-time for home office

## Interview Process

1. **Application review** (async) — Cover letter + GitHub profile review (24h)
2. **Async writing task** (3 days) — Write a 500-word tutorial for one Mekong CLI command
3. **Video interview** (45 min) — Culture fit, role expectations, async work philosophy
4. **Paid trial project** (1 week, $500) — Run Discord for one week, triage 5 GitHub issues
5. **Offer** — 3 business days after trial

**Total process time:** 2–3 weeks
**Hiring decision maker:** Founder only (no committee pre-Series A)
