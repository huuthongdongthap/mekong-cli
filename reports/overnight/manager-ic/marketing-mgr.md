## Target Audience Personas

**Persona 1: "The Indie Hacker"**
- 25-35, solo founder, builds 2-4 projects/year
- Follows: @levelsio, Indie Hackers, r/LocalLLaMA
- Pain: too much repetitive scaffolding, wants AI to "just do it"
- Hook: `mekong cook "Create a SaaS landing page"` → immediate value

**Persona 2: "The Bootstrapped CTO"**
- Small agency CTO (5-15 person team)
- Uses Claude Code daily, wants CLI automation layer
- Hook: 289 commands = replacing 10 different tools
- Upgrade path: Starter → Enterprise within 60 days

**Persona 3: "The Non-Tech Founder"**
- Business background, outsources tech
- Hook: `mekong annual "Kế hoạch kinh doanh 2026"` — zero code required
- High LTV if retained (needs more hand-holding)
