## Brand Essence

**Name:** Mekong CLI
**Tagline:** *"AI-operated business platform. Open source. Universal LLM."*
**War cry:** *"OpenClaw. I run this company."*

The Mekong River flows through 6 countries — relentless, vast, connecting. Mekong CLI connects all 5 business layers into one continuous execution flow. The name signals Southeast Asian roots, global ambition, and unstoppable momentum.

## Positioning Statement

**For** developers and technical founders
**Who** need AI to actually execute business tasks, not just generate advice
**Mekong CLI** is the AI business operating system
**That** covers all 5 layers (Founder → Ops) with a self-healing PEV execution engine
**Unlike** Cursor, Aider, or ChatGPT
**Which** cover only code or offer generation without execution

## Brand Pillars

### 1. Execution Over Advice
Every other AI tool gives you output to act on yourself. Mekong CLI acts.
`mekong cook "build REST API"` → API running. Not instructions. Not a plan. Done.

### 2. Universal, Not Locked
3 env vars. Any LLM. DeepSeek, Qwen, Claude, Ollama local, GPT-4o.
No subscription to a specific model. No vendor tax. Your keys, your models.

### 3. 5 Layers, One Tool
Founder strategy to engineer deployment in a single platform.
`mekong annual` → `mekong plan` → `mekong cook` → `mekong deploy`
The whole company arc, one CLI.

### 4. Binh Pháp Precision
Strategy from 孫子兵法 (Art of War) encoded into software delivery.
Every command is a calculated move. Plan → Execute → Verify = no wasted effort.

### 5. Open Source Trust
MIT license. Read the code. Fork it. Audit it. Own it.
No black boxes. No hidden prompts. Full transparency.

## Voice & Tone

| Context | Tone | Example |
|---------|------|---------|
| README / docs | Direct, technical, confident | "289 commands. 5 business layers. PEV engine." |
| Error messages | Helpful, specific, never condescending | "LLM_API_KEY not set. Export it: export LLM_API_KEY=..." |
| Marketing copy | Bold, minimal, punchy | "Your AI co-founder. In a terminal." |
| Community | Vietnamese warmth + technical depth | Mix VN cultural references with eng precision |
| Investor materials | Data-first, honest about risks | Show gross margins before asking for valuation |

**Never:** Hype without substance. Vague AI buzzwords ("revolutionary", "game-changing"). Overselling before PMF.

### Typography

- **Headers:** JetBrains Mono (monospace, developer credibility)
- **Body:** Inter (clean, readable)
- **Terminal output:** JetBrains Mono, syntax highlighted

### Logo Concept

An ASCII-art claw / terminal prompt hybrid:
```
> mekong_
```
The underscore cursor blinks — always ready to execute. The `>` prompt is universal to every developer.

### README Hero GIF

This GIF is the #1 conversion asset. Every developer who sees it understands instantly.
