# gogcli Secondary Agent (Cửu Biến)
Date: 2026-02-23

## Status
- **Installation**: Verified present via `brew info gogcli` (Version: 0.10.0, Path: `/opt/homebrew/Cellar/gogcli`).
- **Purpose**: High-speed, low-overhead native Golang CLI for Gemini API interactions.

## Binh Pháp Mapping
**九變 CỬU BIẾN (Chín sự biến hóa)**: Phép dùng binh phải linh hoạt tùy theo tình thế.
- **Node.js Overload**: When the main Antigravity Proxy or `claudekit-engineer` (`node`) stack is consuming excessive memory (as seen during extreme compaction), `gogcli` serves as a resilient, compiled alternative without Node.js GC pauses.
- **Role**: Secondary failover agent for simple code generation scripts where full contextual history is unnecessary.
