# LobeHub OpenClaw Command Center (Chính Kỳ)
Date: 2026-02-23

## Status
- **Blueprint Established**: LobeHub (`lobehub/lobe-chat` Docker image) has been identified as the premier visual dashboard capable of loading OpenClaw MCP tools.
- **MCP Bridge**: Requires running the HTTP SSE stream gateway locally (e.g., node proxy -> MCP server).

## Binh Pháp Mapping
**兵勢 CHÍNH KỲ (Lực Lượng & Biến Hóa)**: 
- LobeHub represents **CHÍNH** (The main army base / visual GUI for the CTO/Chairman).
- OpenClaw daemons represent **KỲ** (The dynamic, autonomous execution arms via Telegram, CLI, or autonomous loop).

By bridging them via MCP, the user interacts with standard chat while deploying parallel automation underneath. Test skipped to conserve immediate RAM footprints, mapped out for production.
