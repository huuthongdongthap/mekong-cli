# Exa MCP Search (Dụng Gián)
Date: 2026-02-23

## Status
- **Installation**: Configured via direct MCP Server URL injection for Antigravity Proxy.
- **Proxy Configuration**: `serverUrl` set to `https://mcp.exa.ai/mcp` within `~/.gemini/settings.json`.
- **Environment Context**: Expects `EXA_API_KEY` to be passed or configured by the user via the `dashboard.exa.ai` portal.

## Tool Capabilities
Hosted cloud MCP supports:
- `web_search_exa`
- `get_code_context_exa`
- `company_research_exa`
- `deep_researcher_start`
- `crawling_exa`

## Binh Pháp Mapping
**用間 DÙNG GIÁN** (Gián điệp thông minh): Provides deep, targeted search directly returning LLM-friendly results instead of raw HTML, offering a strategic intelligence advantage outperforming standard web search (Brave).
