## 🎯 How to Use Each Format

### PDF (`raas-sales-deck.pdf`)
**Best for:** Email attachments, printing, client handouts

- ✅ Attach to sales emails
- ✅ Print for in-person meetings
- ✅ Share via Slack/Discord
- ✅ Upload to sales enablement tools

### PowerPoint (`raas-sales-deck.pptx`)
**Best for:** Customizable presentations, team training

- ✅ Import to Google Slides: Google Drive → New → File Upload → Select PPTX
- ✅ Edit in Microsoft PowerPoint
- ✅ Customize for specific client pitches
- ✅ Add company-specific case studies

### HTML (`raas-sales-deck.html`)
**Best for:** Browser presentation, web embedding

- ✅ Open in browser for presentations (F11 for fullscreen)
- ✅ Embed in internal wiki/documentation
- ✅ Convert to PDF via browser (Cmd+P → Save as PDF)

### Print-optimized HTML (`raas-sales-deck-printable.html`)
**Best for:** High-quality PDF via browser

1. Open in browser (Chrome/Safari recommended)
2. Press Cmd+P (Mac) or Ctrl+P (Windows)
3. Select "Save as PDF" as destination
4. Click "Save" → Get beautifully formatted PDF

### Markdown (`raas-sales-deck-unified.md`)
**Best for:** Source editing, documentation, web publishing

- ✅ Edit in VS Code, Obsidian, or any text editor
- ✅ Publish to GitHub/GitLab
- ✅ Convert to other formats via pandoc
- ✅ Version control with git

## 🎨 Customization Guide

### To Edit the Deck

**Prerequisites:**
- pandoc (for format conversion)
- xelatex (for PDF generation)
- PowerPoint or Google Slides (for PPTX editing)

**Install on macOS:**
```bash
brew install pandoc
brew install --cask mactex-no-gui
```

**Regenerate All Formats:**
```bash
# From project root
bash scripts/generate_raas_sales_deck.sh

# PDF is auto-generated if xelatex is available
# PPTX is auto-generated by pandoc
```

**Edit Source Files:**
- Main deck: `sales/raas-sales-deck.md`
- Marketing version: `marketing/raas-sales-deck.md`
- Full pitch: `docs/raas-sales-pitch-v2.md`
- Brochure: `docs/raas-sales-brochure.md`
