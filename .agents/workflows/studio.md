# /studio — VC Studio & Founder Command Suite

**AUTO-EXECUTE MODE.** Detect sub-command from user prompt and execute.

## Studio Commands

### `/studio strategy` — Strategic Planning Session
1. Analyze current portfolio state
2. Market opportunity assessment
3. Resource allocation review
4. Create strategic plan in `reports/studio/strategy/`

### `/studio portfolio` — Portfolio Overview
1. List all active projects/ventures
2. Stage and health of each
3. Revenue/traction summary
4. Recommendations

### `/studio bootstrap` — Bootstrap New Venture
1. Validate business idea (use `/idea` workflow first)
2. Create project scaffold
3. Setup initial codebase structure
4. Generate first sprint plan

### `/studio invest` — Investment Analysis
1. Due diligence on opportunity
2. Market size (TAM/SAM/SOM)
3. Team assessment
4. Financial projection
5. Decision: INVEST / PASS / WATCH

### `/studio divest` — Divestment Planning
1. Valuation assessment
2. Buyer targeting
3. Transition plan
4. Timeline and milestones

### `/studio audit` — Portfolio Audit
1. Review all active ventures
2. Performance vs targets
3. Cash flow analysis
4. Risk assessment per venture

### `/studio report` — Studio Report
1. Executive summary
2. Key metrics dashboard (text)
3. Notable events
4. Next period focus

### `/studio roi` — ROI Analysis
1. Per-venture ROI calculation
2. Time-weighted returns
3. Benchmark comparison
4. Optimization recommendations

### `/studio launch-full` — Full Launch Sequence
1. Pre-launch checklist
2. Product readiness verification
3. Marketing materials  
4. Launch execution
5. Post-launch monitoring

### `/studio operate-daily` — Daily Operations
1. Check each venture status
2. Priority triage
3. Resource reallocation if needed
4. End-of-day summary

### `/studio sprint-weekly` — Weekly Sprint
1. Review last week's progress
2. Plan current week
3. Assign priorities
4. Track blockers

### `/studio diligence-deep` — Deep Due Diligence
1. Technical architecture review
2. Code quality audit
3. Security assessment
4. Scalability analysis
5. Team capability evaluation

### `/studio allocate` — Resource Allocation
1. Current resource map
2. Utilization analysis
3. Rebalancing recommendations

## Founder Commands

### `/founder raise` — Fundraising Workflow
1. Pitch deck structure
2. Financial model template
3. VC targeting list
4. Outreach sequence

### `/founder negotiate` — Negotiation Playbook
1. Term sheet analysis
2. Key terms reference
3. Counter-offer strategy

### `/founder validate` — Idea Validation
1. Customer interviews framework
2. MVP definition
3. Hypothesis testing plan

### `/founder ipo` — IPO Preparation
1. Governance checklist
2. Audit trail requirements
3. Regulatory compliance (VN/SEA)
4. Timeline planning

## Venture Analysis

### `/venture five-factors` — 5-Factor Assessment
Analyze: Market, Team, Product, Traction, Timing

### `/venture terrain` — Market Terrain Mapping
Map competitive landscape, entry barriers, market dynamics

### `/venture thesis` — Investment Thesis
Generate thesis document for the venture

### `/venture momentum` — Momentum Score
Assess growth momentum across key metrics

### `/venture void-substance` — Void & Substance Analysis
Binh Phap strategic assessment: identify voids (gaps) and substance (strengths)

## Output Convention
All reports go to `reports/studio/` or `reports/founder/` directories.
