▸ Mekong CLAUDE.md — OpenClaw biết cách bill
## BILLING RULES — OpenClaw phải follow TRƯỚC KHI execute bất kỳ LLM call nào: 1. Call POST /v1/mcu/check với tenant_id + estimated_cost 2. Nếu balance < cost → STOP, trả về "insufficient_credits", KHÔNG execute 3. Nếu đủ → POST /v1/mcu/lock (lock MCU trước, tránh race condition) SAU KHI execute xong: 4. POST /v1/mcu/confirm với actual_tokens_used 5. Nếu actual < estimated → refund difference tự động ROUTING LOGIC: - Goal có từ "nhanh" / "draft" / "brainstorm" → routing.bulk - Goal có "code" / "api" / "deploy" → routing.standard - Goal có "architect" / "system design" / "full feature" → routing.complex - Còn lại → routing.simple
