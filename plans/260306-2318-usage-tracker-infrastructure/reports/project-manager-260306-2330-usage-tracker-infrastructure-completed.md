## Database Schema

```sql
CREATE TABLE usage_events (
    id SERIAL PRIMARY KEY,
    key_id VARCHAR(255) NOT NULL,
    license_key_hash VARCHAR(64) NOT NULL,
    event_type VARCHAR(50) NOT NULL,        -- 'command' or 'feature'
    event_data JSONB NOT NULL,
    idempotency_key VARCHAR(64) UNIQUE NOT NULL,
    metadata JSONB DEFAULT '{}',
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);
```

**Indexes:** key_id, created_at, event_type, idempotency_key

## Usage Example

```python
from src.usage.decorators import track_command, track_feature
from src.usage.usage_tracker import get_tracker

tracker = get_tracker()

# Track command
await tracker.track_command(
    key_id="key-123",
    command="cook",
    license_key="license-abc"
)

# Track feature
await tracker.track_feature(
    key_id="key-123",
    feature_tag="bmc",
    license_key="license-abc"
)

# Get summary
summary = await tracker.get_usage_summary("key-123", days=30)
```
