## 🚀 How to Use

### Option 1: View in Browser (Quick Preview)

```bash
cd ~/mekong-cli/plans/proposals/visual
open index.html  # macOS
# or
xdg-open index.html  # Linux
# or double-click index.html in Finder/Explorer
```

### Option 2: Generate PNG Screenshots (Professional Delivery)

**Step 1: Install Playwright (first time only)**
```bash
cd ~/mekong-cli/plans/proposals/visual
npx playwright install chromium
```

**Step 2: Generate screenshots**
```bash
node screenshot.js
```

**Output files:**
- `proposal-full.png` - Complete proposal (all sections)
- `01-cover.png` - Cover slide only
- `02-workflow.png` - Workflow diagram only
- `03-pricing-table.png` - Pricing comparison only
- `04-tier-1.png` to `04-tier-5.png` - Each tier card
- `05-cta.png` - CTA section

### Option 3: Share HTML File Directly

Send `index.html` + `styles.css` to client. They can:
- Open in any modern browser
- View interactive animations
- No internet connection required (self-contained)

## 🎨 Design Features

### Color Scheme
- **Primary Gradient:** Purple to Blue (#667eea → #764ba2)

- **Accents:** Cyan/Pink gradients for visual impact

- Fade-in effects on cover slide
