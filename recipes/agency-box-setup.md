# Agency-in-a-Box Setup

This recipe scaffolds a new agency environment in the `clients/` directory.
It expects `CLIENT_NAME` environment variable to be set. Defaults to `demo-agency`.

## Step 1: Prepare Environment
echo "Setting up environment..."
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
echo "Target Client: $CLIENT_NAME"
echo "Target Directory: $TARGET_DIR"

## Step 2: Create Directory Structure
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
mkdir -p "$TARGET_DIR"
mkdir -p "$TARGET_DIR/config"
mkdir -p "$TARGET_DIR/docs"
echo "Created directory structure at $TARGET_DIR"

## Step 3: Scaffold Landing Page
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
echo "Scaffolding landing page from template..."
# Copy agencyos-landing as the base template
# Excluding node_modules and .git
cp -R apps/agencyos-landing "$TARGET_DIR/landing"
rm -rf "$TARGET_DIR/landing/node_modules"
rm -rf "$TARGET_DIR/landing/.next"
rm -rf "$TARGET_DIR/landing/.git"
echo "Landing page scaffolded at $TARGET_DIR/landing"

## Step 4: Configure Deployment
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
echo "Generating Cloudflare Pages configuration..."
cat > "$TARGET_DIR/landing/wrangler.toml" <<EOF
name = "$CLIENT_NAME-landing"
compatibility_date = "2024-01-01"
pages_build_output_dir = ".next"

[build]
command = "npm run build"
EOF
echo "wrangler.toml configured."

## Step 5: Initialize Client Documentation
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
echo "Creating client documentation..."
cat > "$TARGET_DIR/docs/README.md" <<EOF
# $CLIENT_NAME AgencyOS Environment

Generated by Mekong CLI Agency-in-a-Box Recipe.

## Components
- **Landing Page**: ./landing (Next.js)
- **Configuration**: ./config

## Deployment
1. Navigate to ./landing
2. Run \`npm install\`
3. Run \`wrangler pages deploy .next\`
EOF
echo "Documentation initialized."

## Step 6: Final Verification
export CLIENT_NAME="${CLIENT_NAME:-demo-agency}"
export TARGET_DIR="clients/$CLIENT_NAME"
ls -F "$TARGET_DIR"
echo "Agency setup complete for $CLIENT_NAME"
