## Copywriting Principles

### Core Rules

1. **Clarity Over Cleverness** - Clear > creative
2. **Benefits Over Features** - What it means for customer
3. **Specificity Over Vagueness** - Numbers, timeframes
4. **Customer Language** - Mirror their words
5. **One Idea Per Section** - Build logical flow

### Style Rules

- Simple over complex ("Use" not "Utilize")
- Active over passive ("We generate" not "Reports are generated")
- Confident over qualified (Remove "almost," "very")
- Show over tell (Describe outcome, not adverbs)

## Page Structure Framework

### Above the Fold

```markdown
**Headline**: Core value proposition

- "{Achieve outcome} without {pain point}"
- "The {category} for {audience}"
- "Never {unpleasant event} again"

**Subheadline**: Expands, adds specificity

**Primary CTA**: Value-focused

- ✅ "Start Free Trial"
- ❌ "Sign Up"
```

### Core Sections

| Section            | Purpose                    |
| ------------------ | -------------------------- |
| Social Proof       | Logos, stats, testimonials |
| Problem/Pain       | Show understanding         |
| Solution/Benefits  | 3-5 key benefits           |
| How It Works       | 3-4 steps                  |
| Objection Handling | FAQ, comparisons           |
| Final CTA          | Value + risk reversal      |

## CTA Guidelines

**Weak (avoid):**
Submit, Sign Up, Learn More, Click Here

**Strong (use):**

- Start Free Trial
- Get [Specific Thing]
- See [Product] in Action
- Create Your First [Thing]

**Formula:** [Action Verb] + [What They Get] + [Qualifier]
