## Landing Page Frameworks

### AIDA Framework
```
Attention → Interest → Desire → Action
```

### PAS Framework
```
Problem → Agitation → Solution
```

### FAB Framework
```
Features → Advantages → Benefits
```

## Headline Formulas

```bash
# How To + Benefit + Timeframe
"How to automate your agency in 30 minutes"

# Number + Adjective + Keyword + Benefit
"7 powerful automations saving agencies 20hrs/week"

# Question + Agitate + Solution
"Tired of manual ops? Here's the fix."

# Before/After + Bridge
"From chaos to calm: Automate everything"

# Secret/Insider
"What top agencies know about scaling"
```

## CTA Copy Formulas

```bash
# Action Verb + Value
"Start My Free Trial"
"Get Instant Access"
"Claim My Discount"

# Low Commitment
"See How It Works"
"Explore Features"
"Watch Demo"

# Urgency
"Get Offer Before It's Gone"
"Claim Spot — Only 3 Left"
"Start Saving Today"
```

## Conversion Optimization

```yaml
test_headlines:
  variant_a: "Automate Your Agency Operations"
  variant_b: "Never Do Manual Ops Again"
  metric: click_through_rate

  variant_a: "Start Free Trial"
  variant_b: "Get Started Free"
  metric: conversion_rate

test_pricing:
  variant_a: "$99/month"
  variant_b: "$99"
  variant_c: "From $99"
  metric: upgrade_rate

  variant_a: "Trusted by 1,000+ agencies"
  variant_b: "Join 1,000+ happy customers"

```
