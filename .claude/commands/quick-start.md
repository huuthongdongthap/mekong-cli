# 🚀 Quick Start: New Project in 5 Commands

> **"Dễ như ăn kẹo!"** - Any project, 5 commands, done.

---

## The 5 Commands

```
1. /brainstorm "My idea"     → Ideas + validation
2. /plan                     → Implementation plan
3. /code                     → Build it
4. /ship                     → Deploy to production
5. /money                    → Start revenue
```

**That's it!** 🏯

---

## Detailed Flow

### Command 1: `/brainstorm "My SaaS idea"`

**What happens:**

- AI generates 5-10 related ideas
- Binh Pháp strategic analysis
- WIN-WIN-WIN validation

**Output:**

```
✅ Decision: GO
💡 Top ideas: 3 prioritized
📊 Strategy: Market-first approach
```

---

### Command 2: `/plan`

**What happens:**

- Creates PRD document
- Architecture diagram
- Task breakdown

**Output:**

```
📋 PRD: docs/prd/my-saas.md
🏗️ Architecture: Mermaid diagram
📝 Tasks: 12 items in TODO
```

---

### Command 3: `/code`

**What happens:**

- TDD approach (tests first)
- Implementation
- Code quality check

**Output:**

```
✅ Tests: 15 passing
💻 Code: 8 files created
🔍 Quality: 94/100
```

---

### Command 4: `/ship`

**What happens:**

- Run CI locally
- Commit + Push
- Deploy to production

**Output:**

```
✅ CI: All checks pass
✅ Commit: a1b2c3d
🚀 Live: https://my-saas.pages.dev
```

---

### Command 5: `/money`

**What happens:**

- Generate marketing content
- Process leads
- Send outreach

**Output:**

```
✅ 3 leads generated
✅ 2 articles created
💰 Projected MRR: $2,500
```
