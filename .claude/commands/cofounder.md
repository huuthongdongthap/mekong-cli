## Equity Split Framework (Slicing Pie)

```
Base: 50/50 split, then adjust:

+10% for: The original idea + initial validation
+10% for: Full-time commitment (vs part-time)
+10% for: Domain expertise in core area
+10% for: Bringing initial capital
+5%  for: Existing customer relationships
+5%  for: Critical technical skills

Example:
- Founder A (idea + full-time + domain): 60%
- Founder B (capital + technical): 40%
```

## Vesting Schedule Template

```yaml
standard_4year_1cliff:
  total_shares: 1,000,000
  vesting_period: 48 months
  cliff: 12 months  # No shares vest before 1 year
  cliff_vest: 25%   # 25% vests at cliff
  monthly_vest: 2.08% # Remaining vests monthly
  
accelerated_triggers:
  single_trigger: false  # Don't accelerate on acquisition alone
  double_trigger: true   # Accelerate if fired after acquisition
```

## Operating Agreement Checklist

- [ ] Equity split with vesting schedule
- [ ] Roles and responsibilities defined
- [ ] Decision-making process (unanimous vs majority)
- [ ] IP assignment to company
- [ ] Non-compete and non-solicit clauses
- [ ] Departure terms (good leaver vs bad leaver)
- [ ] Dispute resolution mechanism
- [ ] Financial commitment expectations
- [ ] Working hours and location expectations
- [ ] Intellectual property ownership

## Red Flags 🚩

- Wants equity but won't commit full-time
- No relevant skills (just "ideas")
- Unwilling to sign vesting agreement
- History of abandoned projects
- Misaligned risk tolerance
