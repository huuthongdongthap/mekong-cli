## Referral Program Framework

### One-Sided (Referrer Only)

```markdown
Example: "Refer a friend, get $10 credit"

- Lower friction
- Easier to understand
- Good for: B2C, low-ticket
```

### Two-Sided (Both Get Reward)

```markdown
Example: "Give $10, Get $10"

- Higher conversion
- More viral
- Good for: SaaS, subscription
```

### Referral Copy Templates

```markdown
**Invite Message:**
"Hey! I've been using [Product] and thought you'd love it.
Use my link to get [Reward]: [link]"

**Landing Page:**
"[Friend's name] gave you [Reward]"
"Claim your [reward] now"
```

## Free Tool Strategy

### Types of Free Tools

```markdown
1. **Calculators** - ROI calculator, pricing estimator
2. **Generators** - Name generator, template generator
3. **Analyzers** - Website audit, SEO checker
4. **Converters** - File converter, format converter
```

## Psychology Triggers

### 10 Key Triggers

```markdown
1. **Scarcity** - "Only 5 left" / "Limited time"
2. **Urgency** - "Ends tonight" / "Flash sale"
3. **Social Proof** - "10,000+ customers" / "4.9 stars"
4. **Authority** - "Recommended by [expert]"
5. **Reciprocity** - Free value first
6. **Loss Aversion** - "Don't miss out"
7. **Anchoring** - Show higher price first
8. **Bandwagon** - "Join 5,000 others"
9. **Curiosity** - "The secret to..."
10. **Exclusivity** - "VIP access" / "Members only"
```

### Apply to Campaigns

```bash
/marketing-growth psychology --trigger=scarcity --context="launch promotion"
/marketing-growth psychology --trigger=social-proof --context="landing page"
```

## Launch Strategy

### Pre-Launch

```markdown
1. Build waitlist
2. Create anticipation
3. Beta testers / early access
4. Press/influencer outreach
```

### Launch Day

```markdown
1. Email blast to waitlist
2. Social media blitz
3. PR coverage
4. Community engagement
5. Live event/demo
```

### Post-Launch

```markdown
1. Collect testimonials
2. Case studies
3. Retarget non-converters
4. Iterate based on feedback
```
