# Playwright CLI 🎭 (Hư Thực)
Sử dụng Playwright CLI từ npm (`playwright-cli`) để kiểm tra UI, tương tác mạng và đo đạc storage.

## Lệnh Thường Dùng (Sử dụng lệnh trực tiếp trong CC CLI)
- `@snapshot <url>`: Chụp toàn bộ accessibility snapshot.
- `@screenshot <url>`: Chụp ảnh màn hình thật.
- `@click <url> <selector>`: Click theo tọa độ/selector.
- `@fill <url> <selector> "<text>"`: Điền form.
