# @vibe/vibe-ui

Core UI component library for Vibe applications.

## Installation

```bash
pnpm add @vibe/vibe-ui
```

## Usage

```typescript
import { Button, Card } from '@vibe/vibe-ui';
```

## Features

- React components with TypeScript support
- Consistent design system
- Accessible by default

## Development

```bash
pnpm build
pnpm dev
```
