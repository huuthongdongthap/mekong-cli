# Vibe UI Packages

UI-related packages for the Vibe platform.

## Packages

- **@vibe/vibe-ui** - Core UI component library
- **@vibe/i18n** - Internationalization utilities

## Development

```bash
pnpm install
pnpm build
```

## Usage

Import components from individual packages:

```typescript
import { Button } from '@vibe/vibe-ui';
import { t } from '@vibe/i18n';
```
