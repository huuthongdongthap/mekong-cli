# @vibe/shared

Shared utilities and types for Vibe packages.

## Installation

```bash
pnpm add @vibe/shared
```

## Usage

```typescript
import { logger, config } from '@vibe/shared';

logger.info('Application started');
```

## Features

- Common utilities
- Shared TypeScript types
- Configuration helpers
- Logging utilities

## Development

```bash
pnpm build
pnpm dev
```
