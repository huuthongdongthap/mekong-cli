# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added
- **DORA Metrics Engine**: Calculate Deployment Frequency, Lead Time for Changes, and Change Failure Rate.
- **Engineering Velocity Metrics**: Track Cycle Time, PR Pickup Time, PR Review Time, and PR Merge Time.
- **GitHub GraphQL Client**: Optimized data fetching for repositories, PRs, and releases.
- **CLI Integration**: `metricsCommand` for use with `@agencyos/vibe-dev` CLI.
- **Auto-Detection**: Automatically detects repository context from git config.
- **JSON Output**: Support for machine-readable output for CI/CD integration.
- **Telemtry**: Foundation for Growth Telemetry Engine.

### Documentation
- Initial README with usage instructions and API examples.
