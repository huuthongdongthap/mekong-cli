# ApexOS

**Revenue As A Service (RaaS)** - AI agents platform targeting $1M ARR.

## Quick Start

```bash
# Install dependencies
pnpm install

# Development
pnpm dev

# Build
pnpm build

# Test
pnpm test
```

## Stack

- **Frontend:** React 19 + Vite + TailwindCSS
- **Testing:** Vitest + Testing Library
- **Target:** $1M ARR via RaaS model
