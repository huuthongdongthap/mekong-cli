## What
- Claude Code mobile client (iOS + Android + Web)
- Điều khiển CC CLI từ phone, E2E encrypted
- Voice interaction, push notifications, multi-session

## Key Features
- 📱 Mobile control CC CLI from anywhere
- 🔐 E2E AES encryption (relay can't read)
- 🎙️ Voice commands — hands-free
- 🔄 Multiple CC CLI instances parallel
- 🔔 Push notifications when CC CLI needs input
- 🖥️ Web app: app.happy.engineering
