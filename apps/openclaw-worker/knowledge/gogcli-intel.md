## What
- Full Google Suite in terminal: Gmail, Calendar, Drive, Sheets, Docs, Tasks, Chat, Contacts, Forms, Apps Script, Classroom, Keep, Groups
- JSON-first output = AI-friendly
- Multi-account, least-privilege auth, command sandboxing
- By steipete (Peter Steinberger, PSPDFKit founder)

## Install
```bash
brew install gogcli
gog auth add your@email.com
```
