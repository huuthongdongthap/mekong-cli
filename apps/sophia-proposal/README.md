# Sophia AI Factory

> **Status:** ✅ GREEN Production | **Version:** 1.0.0 | **Last Updated:** 2026-03-12

Sophia Proposal - Landing page và AI Video Factory platform với affiliate marketing integration.

## Quick Start

```bash
# Install
pnpm install

# Development
pnpm dev

# Build & Test
pnpm run build && pnpm vitest run

# Deploy
pnpm run deploy:cf
```
