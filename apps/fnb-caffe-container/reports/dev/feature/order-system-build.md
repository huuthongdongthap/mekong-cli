## API Endpoints

### Cart API
```
POST /api/cart/add          - Thêm sản phẩm vào giỏ
POST /api/cart/update       - Cập nhật số lượng
POST /api/cart/remove       - Xóa sản phẩm
GET  /api/cart              - Lấy giỏ hàng
POST /api/cart/clear        - Xóa toàn bộ giỏ
```

### Checkout API
```
POST /api/checkout          - Tạo đơn hàng mới
GET  /api/orders/{id}       - Lấy thông tin đơn hàng
GET  /api/orders            - Lấy danh sách đơn hàng
```

### Payment API
```
POST /api/payment/create-url  - Tạo URL thanh toán
GET  /api/payment/vnpay/callback - VNPay callback
GET  /api/payment/momo/callback  - MoMo callback
POST /api/payment/momo/ipn       - MoMo IPN
```
