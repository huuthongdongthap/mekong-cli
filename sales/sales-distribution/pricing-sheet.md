## MCU (Mission Credit Unit) Pricing

### Standard MCU Rates

| MCU Type | Base Cost | Description |
|----------|-----------|-------------|
| **MCU-S (Simple)** | 1 credit | Single-file changes, bug fixes |
| **MCU-M (Medium)** | 3 credits | Multi-file features, integrations |
| **MCU-L (Large)** | 5 credits | Full modules, system changes |
| **MCU-XL (Extra Large)** | Custom | Enterprise migrations, quoted separately |

### Custom MCU Pricing (Enterprise)

For missions exceeding 10 credits:

```
Base Cost = (Estimated Hours × Hourly Rate) / Credit Value

Where:
- Estimated Hours: Time for human developer
- Hourly Rate: $75 (US average developer)
- Credit Value: $1.00 (Master tier rate)

Example: 20-hour migration = (20 × $75) / $1 = 1,500 credits
Enterprise quote: 1,200 credits (20% volume discount)
```

## ROI Calculator

### Agency Savings Formula

```
Monthly Savings = (Hours Saved × Hourly Rate) - Tier Cost

Where:
- Hours Saved = Missions/Month × Average Hours/Mission
- Hourly Rate = Blended developer rate ($50-150)
- Tier Cost = Monthly subscription
```
