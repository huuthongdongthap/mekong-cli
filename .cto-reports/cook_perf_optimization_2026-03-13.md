### 5. Cache Optimization

**Vercel.json** (đã sẵn):
```json
{
  "/assets/(.*)": "public, max-age=31536000, immutable",
  "/(css|js)/(.*)": "public, max-age=31536000, immutable",
  "/images/(.*)": "public, max-age=2592000, stale-while-revalidate",
  "/fonts/(.*)": "public, max-age=31536000, immutable"
}
```

**Service Worker** (đã sẵn):
- Cache-first strategy cho static assets
- Network-first strategy cho API calls
- Stale-while-revalidate cho HTML pages
