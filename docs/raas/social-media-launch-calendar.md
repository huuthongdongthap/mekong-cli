## Engagement Guidelines

### Best Practices

| Do | Don't |
|----|-------|
| Respond to every comment in first 2 hours | Argue with critics publicly |
| Tag relevant people/companies | Spam hashtags |
| Share behind-the-scenes | Over-promise features |
| Use thread format for depth | Post walls of text on LinkedIn |
| Include visuals (screenshots, GIFs) | Post text-only on Twitter |

| Platform | Time (PST) | Day |

| LinkedIn | 8-10 AM | Tue-Thu |
| Twitter | 12-1 PM | Daily |
| Twitter | 8-9 AM | Daily |
