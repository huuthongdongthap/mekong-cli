## Upgrade Path

### How to Upgrade

1. **Free → Starter/Pro**: Self-service in dashboard (Settings → Billing)
2. **Starter/Pro → Enterprise**: Contact sales (enterprise@agencyos.network)
3. **Enterprise → Custom**: Account manager handles package changes

### Upgrade Timing

| Upgrade Type | Effective Date | Proration |
|--------------|----------------|-----------|
| Free → Starter | Immediate | Yes (full month credit) |
| Free → Pro | Immediate | Yes (full month credit) |
| Starter → Pro | Immediate | Yes (unused credits converted) |
| Any → Enterprise | Contract start | Custom billing |

### Downgrade Policy

- Downgrade anytime in dashboard
- Effective at next billing cycle
- Unused credits forfeited (except Pro rollover)
- 30-day grace period to restore previous plan

## FAQ

**Q: Can I switch plans anytime?**

A: Yes! Upgrade anytime in dashboard. Downgrades take effect next billing cycle.

**Q: What happens if I run out of credits?**

A: You can purchase extra credits or wait until next month's reset. Overage charges apply automatically.

**Q: Do credits expire?**

A: Monthly credits reset on your billing date. Pro tier: rolled credits expire after 90 days.

**Q: Can I pause my subscription?**

A: Yes, pause up to 3 months/year. Paused subscriptions retain data but cannot run missions.

**Q: What's included in "priority support"?**

A: Pro: 4-hour email response. Enterprise: 15-minute P0 escalation + dedicated Slack channel.

**Q: Do you offer nonprofit discounts?**

A: Yes! 50% off for registered 501(c)(3) organizations. Contact: social@agencyos.network

**Q: Can I get a custom plan?**

A: Enterprise customers can request custom credit packages, RTO/RPO, and feature bundles.
