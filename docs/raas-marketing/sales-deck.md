## Slide 19: Appendix — Mission Lifecycle

# Mission Execution Flow

```
┌────────────────────────────────────────────────────────────┐
│  1. SUBMIT                                                 │
│  POST /v1/missions                                         │
│  {"goal": "...", "complexity": "standard"}                 │
└─────────────┬──────────────────────────────────────────────┘
              │
              ▼
┌────────────────────────────────────────────────────────────┐
│  2. VALIDATE                                               │
│  • Auth check (JWT/API key)                                │
│  • Credit balance check                                    │
│  • Input validation                                        │
└─────────────┬──────────────────────────────────────────────┘
              │
              ▼
┌────────────────────────────────────────────────────────────┐
│  3. PLAN                                                   │
│  LLM decomposes goal into steps                            │
│  Output: ["Step 1: ...", "Step 2: ...", ...]               │
└─────────────┬──────────────────────────────────────────────┘
              │
              ▼
┌────────────────────────────────────────────────────────────┐
│  4. EXECUTE                                                │
│  Run each step via Shell/LLM/API                           │
│  Real-time logging to SSE stream                           │
└─────────────┬──────────────────────────────────────────────┘
              │
              ▼
┌────────────────────────────────────────────────────────────┐
│  5. VERIFY                                                 │
│  Quality gates: build, test, lint                          │
│  Rollback if verification fails                            │
└─────────────┬──────────────────────────────────────────────┘
              │
              ▼
┌────────────────────────────────────────────────────────────┐
│  6. COMPLETE                                               │
│  Status: completed/failed                                  │
│  Deduct credits, notify user                               │
└────────────────────────────────────────────────────────────┘
```
