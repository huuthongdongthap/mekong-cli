# 30-Day Go-To-Market Launch Checklist — Mekong CLI / OpenClaw RaaS

**Goal:** $5k revenue by Day 30. 200+ PH upvotes. 5 enterprise pilot conversations started.

---

## Week 1: Pre-Launch Prep (Days 1-7)

**KPIs:** npm published, all visual assets done, 50 supporters queued, PH listing drafted.

- [ ] **Day 1** — `pip install mekong-cli` works cleanly from PyPI; verify with fresh venv
- [ ] **Day 1** — Run `make self-test` → score 100/100; all tests green
- [ ] **Day 2** — Create all visual assets per [producthunt-launch-kit.md](producthunt-launch-kit.md#visual-assets-checklist): hero screenshot, GIF demo, architecture diagram, logo (240x240, 1270x760), social image (1200x630)
- [ ] **Day 2** — Record 60s terminal demo GIF: `mekong cook "Create REST API with auth"`
- [ ] **Day 3** — Draft PH listing: tagline A ("Run your entire company from 319 CLI commands"), description, maker comment — see [producthunt-launch-kit.md](producthunt-launch-kit.md#ph-listing-description-260-chars)
- [ ] **Day 3** — Schedule PH launch for Day 8 (Tuesday or Wednesday, 12:01 AM PT)
- [ ] **Day 4** — Prep Twitter thread (5 tweets) + LinkedIn post per [producthunt-launch-kit.md](producthunt-launch-kit.md#twitterx-thread-5-tweets)
- [ ] **Day 4** — Build supporter list: 50 friends/colleagues with PH accounts; pre-DM with date
- [ ] **Day 5** — Setup landing page pricing: Starter $49, Pro $149, Enterprise $499 on Polar.sh
- [ ] **Day 5** — Enable pre-order on Polar.sh (annual plans, early-bird 20% discount)
- [ ] **Day 6** — Post teaser on Twitter/LinkedIn: "Launching something next week..."
- [ ] **Day 6** — Join 5 relevant communities: IndieHackers, r/SideProject, r/startups, CLI Discords, VN dev groups
- [ ] **Day 7** — Final dry run: all links work, GIFs load, pricing page live, PH listing preview sent to 5 friends

**Revenue target:** $0 (pre-launch)

---

## Week 2: ProductHunt Launch (Days 8-14)

**KPIs:** 200+ upvotes, 500+ website visits, $500 in pre-orders/starter plans.

### Launch Day (Day 8) — follow [producthunt-launch-kit.md](producthunt-launch-kit.md#launch-day-schedule)

- [ ] T-12h: Final check all assets
- [ ] T-1h: Send preview to close friends for early comments
- [ ] T-0: Launch! Post maker comment immediately
- [ ] T+5m: Post first comment (technical details)
- [ ] T+15m: Publish Twitter thread
- [ ] T+30m: Publish LinkedIn post
- [ ] T+1h: DM 50 supporters with direct PH link
- [ ] T+2h: Reply to every PH comment
- [ ] T+4h: Share in Discord/Slack communities
- [ ] T+8h: Post update with traction numbers
- [ ] T+24h: "Lessons learned" tweet

### Post-Launch (Days 9-14)

- [ ] **Day 9** — Reply ALL remaining PH comments; share final PH stats on social
- [ ] **Day 10** — Publish blog post: "Building an AI CTO — technical deep dive"
- [ ] **Day 11** — Submit to HackerNews (Show HN)
- [ ] **Day 12** — Reach out to 5 tech bloggers/YouTubers for reviews
- [ ] **Day 13** — Analyze signup data: top segments, conversion funnel, drop-off points
- [ ] **Day 14** — Weekly retro: PH rank, upvotes, signups, revenue; plan Week 3 outreach

**Revenue target:** $500 (pre-orders + starter plans)

---

## Week 3: Enterprise Outreach (Days 15-21)

**KPIs:** 20 cold emails sent, 5 demo calls booked, 2 pilot agreements signed.

Follow [enterprise-pilot-campaign.md](enterprise-pilot-campaign.md) for ICP, email templates, objection handling.

- [ ] **Day 15** — Build prospect list: 20 targets across 5 ICP segments (digital agencies, dev shops, marketing agencies, consulting firms, startup studios) — see [enterprise-pilot-campaign.md](enterprise-pilot-campaign.md#ideal-customer-profile-icp)
- [ ] **Day 16** — Send Email 1 (Problem Awareness) to all 20 prospects — use [template](enterprise-pilot-campaign.md#email-1-problem-awareness-day-0)
- [ ] **Day 17** — LinkedIn warm outreach: 10 DMs to CTOs/VPs using [warm template](enterprise-pilot-campaign.md#warm-outreach-linkedin-dm)
- [ ] **Day 18** — Follow up on PH leads who signed up but didn't convert; offer demo
- [ ] **Day 19** — Send Email 2 (Social Proof) to non-responders — use [template](enterprise-pilot-campaign.md#email-2-social-proof-day-3)
- [ ] **Day 20** — Book and run first demo calls; use 30-second pitch + 10-min terminal demo
- [ ] **Day 21** — Send Email 3 (Scarcity) to remaining non-responders — use [template](enterprise-pilot-campaign.md#email-3-scarcity--offer-day-7)

**Revenue target:** $1,000 (pilot deposits: 2 x $500)

---

## Week 4: Follow-Up & Conversion (Days 22-30)

**KPIs:** 3+ pilots started, $5k total revenue, conversion rate tracked.

- [ ] **Day 22** — Follow up all demo calls within 24h; send ROI calculator per [enterprise-pilot-campaign.md](enterprise-pilot-campaign.md#roi-calculator)
- [ ] **Day 23** — Onboard first pilot customers: CLI installed, team trained, 5 core commands configured — see [pilot structure](enterprise-pilot-campaign.md#30-day-pilot-structure)
- [ ] **Day 24** — Setup dedicated Slack channels for each pilot; SLA: <4h response
- [ ] **Day 25** — Second outreach wave: 10 more prospects from PH/HN signups
- [ ] **Day 26** — Publish case study from PH launch metrics (upvotes, signups, early usage)
- [ ] **Day 27** — Run AppSumo LTD deal if PH traction warrants it
- [ ] **Day 28** — Week 1 check-in call with pilot customers; track hours saved
- [ ] **Day 29** — Compile metrics report: total signups, conversion %, MRR, pilot feedback
- [ ] **Day 30** — 30-day retro: what worked, what didn't, plan Month 2

**Revenue target:** $5,000 cumulative ($500 pre-orders + $1k starter plans + $1k-2.5k pilot fees + conversions)
