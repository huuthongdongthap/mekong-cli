## Search Tips

**To find a command:**
1. Identify your role (Founder, Business, Product, Engineering, Ops)
2. Open [Command Catalog](./command-catalog.md)
3. Find your layer section
4. Search for the command

**To understand the system:**
1. Read [Codebase Summary](./codebase-summary.md) for overview
2. Explore [ARCHITECTURE.md](./ARCHITECTURE.md) for details
3. Check [API Reference](./api-reference.md) for endpoints

**To integrate via API:**
1. Open [API Reference](./api-reference.md)
2. Find your endpoint group
3. Copy example curl request
4. Execute with your API key
